# Loan Comparison Table Application

A web application for comparing education loan options across different banks based on student profiles and requirements.

## Features

- Interactive loan comparison table
- Multi-criteria filtering (gender, loan amount, loan type, country, university)
- Real-time bank recommendations
- Comprehensive test suite
- Strategic loan amount analysis

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Testing:** Jest, Custom test frameworks
- **Data:** JSON-based bank and loan configurations

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The server will start on `http://localhost:3000`

## Project Structure

- `index.html` - Main application page
- `app.js` - Core application logic
- `server.js` - Express server
- `loan-table-component.js` - Table component
- `test-combinations-*.cjs` - Test suite files
- `reference-combinations-*.json` - Reference test data
