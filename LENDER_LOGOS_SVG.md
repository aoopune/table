## Suggested next step

When you add logos to the repo:

1. Create a folder, e.g. `assets/logos/` or `logos/`.
2. Save each SVG as `{lender-slug}.svg` (e.g. `axis-bank.svg`, `state-bank-of-india.svg`) using the direct URLs or downloads from the pages above.
3. Reference them in your app (e.g. `logos/axis-bank.svg`) or map lender id → path in a small config/manifest.

File generated so you can add these SVGs later.
